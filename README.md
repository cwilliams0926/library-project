# library-project

A simple web app that creates a library of book objects.

# How it's made

**Tech Used**: HTML, CSS, JavaScript

A table was created to showcase the library to the user. Basic
styling was added to make the table readable. The books are stored
in an array called myLibrary. The books themselves are made using
an object constructor. A function called displayBooks() turns
the library array into table rows on the page. There's a modal
dialog section that opens upon clicking a button. This dialog contains
a form that allows the user to add a book to the library. Buttons
are added onto each row to remove a book or change its read status.

# Lessons learned

This project put the new things I learned about objects into practice,
on top of introducing me to the idea of dialogs, using event.preventDefault(), and
other new concepts.
